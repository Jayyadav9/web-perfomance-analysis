# 🌐 Website Performance Analysis

## 📌 Project Overview

This project analyzes website performance using Python to uncover actionable insights from website analytics data. The analysis focuses on user behavior, traffic sources, engagement metrics, session trends, and channel performance through data cleaning, exploratory data analysis (EDA), and data visualization.

The project aims to help businesses understand how users interact with their website, identify the most effective marketing channels, and make data-driven decisions to improve traffic, engagement, and overall website performance.

---

## ❓ Business Questions Addressed

1. **What patterns or trends can be observed in website sessions and users over time?**

2. **Which marketing channel brought the highest number of users to the website, and how can this insight be used to improve traffic from other sources?**

3. **Which channel has the highest average engagement time, and what does that indicate about user behavior and content effectiveness?**

4. **How does the engagement rate vary across different traffic channels?**

5. **Which channels are driving more engaged sessions compared to non-engaged sessions, and what strategies can improve engagement in underperforming channels?**

6. **At what hours of the day does each marketing channel drive the most traffic?**

7. **Is there any correlation between high traffic (sessions) and high engagement rate over time?**
